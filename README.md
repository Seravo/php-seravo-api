# PHP Seravo API

A simple, object oriented wrapper for Seravo API, written with PHP.

Uses [Order Module API](https://api.seravo.com/order/docs#/) & [Public Module API](https://api.seravo.com/public/docs#/).

## Requirements

- PHP >= 8.2

The following environment variables are required to be set before using the library:

- `SERAVO_API_URL`
- `SERAVO_API_CLIENT_ID`
- `SERAVO_API_SECRET`
- `SERAVO_KEYCLOAK_PROVIDER_URL`
- `SERAVO_KEYCLOAK_TOKEN_ENDPOINT_URL`

These values must be set in `/.env` file. See `.env.example`.
